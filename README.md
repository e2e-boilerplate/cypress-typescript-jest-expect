# Cypress Boilerplate
    
Cypress end-to-end test automation boilerplate, Using TypeScript, Jest and Expect.
    
## Getting Started
    	1. git clone git@github.com:e2e-boilerplate/cypress-typescript-jest-expect.git
    	2. cd cypress-typescript-jest-expect
    	3. npm install
    	4. npm run test
